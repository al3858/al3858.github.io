Data Pipeline for Housing Price Forecasting

Drexel University · Jan – March 2026

Apache Spark Python Linear Regression Gradient Boosted Trees Feature Engineering Zillow HVI

Overview

Built a Spark-based machine learning pipeline to forecast next-month housing prices using Zillow's Housing Value Index, engineering lag features and evaluating multiple regression models across U.S. states.

What I Did

  • Developed a Spark-based pipeline to analyze and forecast housing prices using Zillow's HVI dataset.
  • Transformed large time-series datasets from wide to long format using Spark window functions.
  • Engineered lag features at 1-month, 3-month, and 12-month horizons.
  • Built Linear Regression and Gradient Boosted Trees models to predict next-month prices.
  • Evaluated models using RMSE and MAE across multiple U.S. states.
